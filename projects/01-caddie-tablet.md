# P01: Caddie App Tablet UX

## 메타 데이터
- 기간: 2022.03 - 2023.05
- 회사: Golfzon & Golfzon CM
- 역할: Product / UXUI Design
- 플랫폼: Android Tablet, Web Admin
- 타겟 유저: 골프장 캐디 및 관제 운영자

## 1. 프로젝트 요약
골프장 현장에서 캐디가 태블릿으로 라운드 정보를 빠르게 확인하고 조작할 수 있도록 설계한 프로젝트입니다.

## 2. 티맵 연관성 (Why TMAP)
현장의 사용 맥락을 이해하고, 이동 환경(카트)에서 빠른 정보 인지와 상황 판단을 할 수 있는 디바이스/태블릿 UX 설계 경험입니다.

## 3. 핵심 문제 (Problem)
골프장 현장에서는 사용자가 라운드 중 필요한 정보를 빠르게 확인하고 조작해야 했습니다. 카트 이동, 앞/뒤 팀 간격 등 동시다발적인 상황 속에서 무전을 대체하고 실시간 정보를 소화해야 하는 정보 밀도의 문제가 있었습니다.

## 4. 해결 전략 (Strategy)
태블릿 환경의 정보 밀도, 조작 흐름, 현장 사용 맥락을 기준으로 화면 구조와 디자인 가이드를 정리했습니다. 캐디의 라운드 준비부터 종료까지의 여정(Journey Map)을 도출하여 주요 기능을 직관적으로 시각화했습니다.

## 5. 나의 기여 (Ownership)
시장 분석, 경쟁사 벤치마킹, 스타일 가이드 제작, 디자인 시스템 가이드 구축, QA, UT, 기능 개선, Figma 프로토타이핑을 수행했습니다. (Android Tablet 앱 UI/UX 80%, 관제 어드민 웹 100%)
