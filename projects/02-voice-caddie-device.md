# P02: Voice Caddie Device UX

## 메타 데이터
- 기간: 2019.11 - 2021.09
- 회사: Voice Caddie
- 역할: GUI / App Design
- 플랫폼: Smart Watch, Native App (iOS/Android)
- 타겟 유저: 필드에서 워치와 앱을 연동해 사용하는 4050 골퍼

## 1. 프로젝트 요약
스마트워치, 핸드 디바이스, 모바일 앱에서 사용자가 필요한 정보를 빠르게 확인할 수 있도록 GUI와 인터랙션을 설계한 프로젝트입니다.

## 2. 티맵 연관성 (Why TMAP)
매우 제한된 화면(스마트워치)에서 시인성과 정보 위계를 설계한 경험은 차량 내 모바일 환경 및 HUD/클러스터 화면 설계 역량과 직결됩니다. 또한 블루투스 페어링 과정 등 IoT 디바이스와 앱을 연동하는 과정에서의 인지 부하 감소 경험입니다.

## 3. 핵심 문제 (Problem)
디바이스 화면은 제한된 공간 안에서 사용자가 필요한 정보를 빠르게 이해하고 조작할 수 있어야 했습니다. 워치라는 좁은 디스플레이, 하드웨어 센서 데이터를 가공해야 하는 앱 환경, 그리고 익숙하지 않은 4050 타겟 사용자의 블루투스 연동 과정 등 물리적 제약이 많았습니다.

## 4. 해결 전략 (Strategy)
스마트워치, 핸드 디바이스, 모바일 앱 환경에 맞춰 정보 우선순위, GUI 가이드, 인터랙션 방향을 정리했습니다. 연결 대기 시간은 모션 UI로 시각적 피드백을 제공하고, 데이터는 직관적인 통계 시각화로 재구성했습니다.

## 5. 나의 기여 (Ownership)
MySwingCaddie 앱 메인 디자인 및 출시, MyVoiceCaddie 모션/인터랙션 작업, T시리즈 워치 GUI 개발 및 가이드 배포, Y1 Golf GUI 메인 화면 디자인을 수행했습니다.
