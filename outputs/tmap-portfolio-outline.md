## 3. Selected Projects

- **기간**: 2022.03 - 2023.05
- **역할**: Product / UXUI Design
- **요약**: 골프장 현장에서 캐디가 태블릿으로 라운드 정보를 빠르게 확인하고 조작할 수 있도록 설계한 프로젝트

- **기간**: 2019.11 - 2021.09
- **역할**: GUI / App Design
- **요약**: 스마트워치, 핸드 디바이스, 모바일 앱에서 사용자가 필요한 정보를 빠르게 확인할 수 있도록 GUI와 인터랙션을 설계한 프로젝트

- **기간**: 2024.07 - 현재
- **역할**: Product Design
- **요약**: 상담 운영 흐름과 SaaS 핵심 기능을 사용자 행동 단위로 재정리한 프로젝트

### Project 04. Dashboard / Statistics UX
